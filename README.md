# Tetris with Ghost Piece

웹 브라우저와 모바일에서 실행 가능한 테트리스 게임입니다.  
이 프로젝트는 기본 테트리스에 다음 기능을 추가했습니다:

## 기능
- **Ghost Piece**: 현재 블록이 떨어질 위치를 반투명으로 표시
- **세로 경계선**: 블록 양끝에서 바닥까지 세로선 표시, 착지 위치 예측 가능
- **홀드(Hold)**: E 키로 블록을 임시 저장
- **점수 & 레벨**: 줄 삭제 시 점수 증가, 레벨 상승 시 낙하 속도 증가
- **새 블록 P 추가**: 기본 7개 블록 외 추가 블록 포함
- **간단 사운드 효과**: 회전, 고정, 줄 삭제 시 사운드 발생

## 조작
- 화살표 ← → : 좌/우 이동
- 화살표 ↓ : 소프트 드롭
- 화살표 ↑ : 회전
- 스페이스바 : 하드 드롭
- E : 홀드
- R : 게임 재시작
